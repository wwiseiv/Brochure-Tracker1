# BrochureTracker - Complete Platform Overview

**PCBancard Field Sales Platform**  
*Mobile-First Progressive Web App for Sales Excellence*

---

## Executive Summary

BrochureTracker is a comprehensive mobile-first Progressive Web App (PWA) designed specifically for PCBancard field sales representatives. The platform transforms the way sales teams track video brochure deployments, manage merchant relationships, and close deals through AI-powered tools.

**Key Value Propositions:**
- Track physical marketing assets (video brochures) with GPS and QR codes
- 14-stage CRM pipeline for complete deal lifecycle management
- AI-powered sales coaching, prospecting, and document analysis
- Team management with role-based access control
- Offline-capable PWA that works anywhere

## Core Features

### 1. Brochure Drop Management

**What it does:** Tracks the deployment and pickup of video brochures at merchant locations.

**Capabilities:**
- **QR Code Scanning** - Scan brochure IDs using device camera
- **GPS Location Capture** - Automatically records where brochures are deployed
- **Business Information** - Capture merchant name, contact, and details
- **Voice Notes** - Record voice memos that are AI-transcribed and summarized
- **Photo Capture** - Take photos of the business location
- **Pickup Scheduling** - Set dates for brochure retrieval with automatic reminders
- **Lead Scoring** - AI analyzes voice notes to score lead quality (1-100)

**Pages:** Dashboard, New Drop, Scan, Drop Detail, History

## Technical Architecture

### Frontend
- **React 18** with TypeScript
- **Vite** build system
- **Tailwind CSS** for styling
- **shadcn/ui** component library
- **Wouter** for routing
- **TanStack Query** for data fetching

### Backend
- **Express.js** with TypeScript
- **PostgreSQL** database
- **Drizzle ORM** for database access
- **Replit Auth** (OpenID Connect)
- **Google Cloud Storage** for files

### AI/ML Services
- **xAI Grok-4** - Prospect finder with web search
- **Anthropic Claude** - Proposals, chatbot, analysis
- **Google Gemini** - Transcription, summarization, role-play
