# PCBancard Statement Analyzer

AI-powered merchant processing statement analyzer that helps sales agents identify savings and win deals.

## Features

- **Statement Analysis**: Enter merchant's processing data to calculate true interchange costs
- **Savings Calculator**: Compare current fees to PCBancard pricing (Interchange Plus & Dual Pricing)
- **Red Flag Detection**: Automatically identifies overcharges, junk fees, and issues
- **AI-Powered Insights**: Claude analyzes statements and provides custom recommendations
- **Talking Points Generator**: Complete negotiation scripts, objection handlers, and closing statements
- **Copy-Ready Scripts**: One-click copy for all talking points

## Quick Start

### 1. Deploy to Replit

1. Create a new Replit project
2. Upload all files from this package
3. Add your Anthropic API key to Secrets:
   - Key: `VITE_ANTHROPIC_API_KEY`
   - Value: Your API key from console.anthropic.com
4. Click "Run"

### 2. Local Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Usage

### For Sales Agents

1. **Enter Statement Data**
   - Monthly processing volume
   - Number of transactions
   - Total fees from statement
   - Merchant type (retail, restaurant, etc.)
   - Current processor name

2. **Review Analysis**
   - See true interchange cost vs. what merchant pays
   - Identify processor markup percentage
   - View PCBancard pricing comparison
   - Check red flags and issues found

3. **Use Talking Points**
   - Copy opening statement with specific numbers
   - Use discovery questions to uncover pain points
   - Handle objections with proven scripts
   - Close with confidence using the closing statement

## PCBancard Pricing

### Interchange Plus
- Wholesale interchange + 0.20% + $0.10/txn
- $10/month account fee
- No PCI fees, no statement fees, no batch fees

### Dual Pricing
- 3.99% service fee charged to card users
- Merchant pays ~$45/month total
- Effective rate near 0%
