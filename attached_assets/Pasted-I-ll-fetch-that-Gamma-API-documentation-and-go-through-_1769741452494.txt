### AVAILABLE ENDPOINTS

#### 1. GENERATE API (Create from scratch)
**POST /generations**

Required parameters:
- `inputText` (string): The content to generate from. Can be a few words or up to 100,000 tokens (~400,000 chars). Use `\n---\n` for manual card breaks.

Optional parameters:
- `textMode`: "generate" | "condense" | "preserve"
  - generate: Rewrites and expands content
  - condense: Summarizes content
  - preserve: Keeps exact text, may add structure
  
- `format`: "presentation" | "document" | "social" | "webpage"

- `themeId` (string): Theme ID from GET /themes endpoint or copied from app

- `numCards` (integer): 1-60 for Pro, 1-75 for Ultra

- `cardSplit`: "auto" | "inputTextBreaks"
  - auto: Divides by numCards
  - inputTextBreaks: Splits at `\n---\n` markers (ignores numCards)

- `additionalInstructions` (string, 1-500 chars): Extra generation guidance

- `folderIds` (array of strings): Folder IDs to store gamma in

- `exportAs`: "pdf" | "pptx" (generates downloadable file URL)

- `textOptions` (object):
  - `amount`: "brief" | "medium" | "detailed" | "extensive"
  - `tone` (string, 1-500 chars): e.g., "professional, inspiring"
  - `audience` (string, 1-500 chars): e.g., "executives, decision-makers"
  - `language` (string): ISO code like "en", supports 60+ languages

- `imageOptions` (object):
  - `source`: "aiGenerated" | "pictographic" | "stock"

#### 3. GET GENERATION STATUS
**GET /generations/{generationId}**

Poll at ~5 second intervals. Response statuses:
- "pending": Still processing
- "completed": Success - includes gammaUrl and export URLs
- "failed": Error occurred

Success response includes:
- `gammaUrl`: Link to view/edit the gamma
- `pdfUrl` / `pptxUrl`: Download links (temporary - download immediately)
- `credits`: { deducted, remaining }

#### 4. LIST THEMES
**GET /themes**

Query parameters:
- `limit` (integer): Results per page
- `cursor` (string): Pagination cursor
- `search` (string): Filter by name

Response includes:
- `id`, `name`, `type` ("standard" | "custom")
- `colorKeywords`, `toneKeywords`
- `hasMore`, `nextCursor` for pagination

#### 5. LIST FOLDERS
**GET /folders**

Same pagination parameters as themes.
Response includes folder `id` and `name`.
