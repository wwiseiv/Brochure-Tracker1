## The 5 Destination Beliefs

Based on "The Salesperson's Secret Code" - research from 1,000+ salespeople:

| Day | Belief | Key Insight |
|-----|--------|-------------|
| Sunday/Monday | **Fulfilment** | Top performers are 62% desire-driven, 38% fear-driven |
| Tuesday | **Control** | Own your outcomes - internal locus beats external |
| Wednesday | **Resilience** | Stress is good; insufficient recovery is the problem |
| Thursday | **Influence** | Be a guerrilla (flexible), not a gorilla (forceful) |
| Friday/Saturday | **Communication** | Ethos ‚Üí Pathos ‚Üí Logos (credibility ‚Üí emotion ‚Üí logic) |

## Installation

### Step 1: Add the Database Schema

Run this SQL in your PostgreSQL database:

```bash
psql $DATABASE_URL -f database/daily-edge-schema.sql
```

### Step 2: Add the Route File

Copy `server/routes/daily-edge.js` to your project.

### Step 3: Update Your Main Server File

Add these lines to your `server/index.js`:

```javascript
// At the top with other requires
const dailyEdgeRoutes = require('./routes/daily-edge');

// After other route definitions
app.use('/api/daily-edge', dailyEdgeRoutes(pool));
```

### Step 4: Seed the Content

```bash
curl -X POST http://localhost:3001/api/daily-edge/content/bulk \
  -H "Content-Type: application/json" \
  -d @database/daily-edge-seed.json
```

## API Endpoints

### Get Today's Content
```bash
GET /api/daily-edge/today?user_id=1
```

Returns:
- Today's belief (rotates daily)
- Quote, insight, challenge, iconic story for that belief
- User's current streak
- Suggested role-play scenario

### Get Belief Overview
```bash
GET /api/daily-edge/beliefs
```

Returns summary of all 5 beliefs with sample content.

### Get Content by Belief
```bash
GET /api/daily-edge/belief/resilience?content_type=challenge
```

### Mark Content Viewed
```bash
POST /api/daily-edge/view
Body: { "user_id": 1, "content_id": 5, "rating": 5 }
```

### Complete Challenge
```bash
POST /api/daily-edge/challenge/complete
Body: { "user_id": 1, "content_id": 12, "reflection": "I tried this today and..." }
```

### Get User Stats
```bash
GET /api/daily-edge/stats/1
```

### Get Leaderboard
```bash
GET /api/daily-edge/leaderboard?type=streak&limit=10
```

### Add Custom Content
```bash
POST /api/daily-edge/content
Body: {
  "belief": "resilience",
  "content_type": "challenge",
  "title": "The 5-Minute Reset",
  "content": "After your next rejection...",
  "source": "PCBancard Coaching"
}
```

## Frontend Integration

### React Component Example

```jsx
import React, { useState, useEffect } from 'react';

function DailyEdge({ userId }) {
  const [content, setContent] = useState(null);
  const [loading, setLoading] = useState(true);

  useEffect(() => {
    fetch(`/api/daily-edge/today?user_id=${userId}`)
      .then(res => res.json())
      .then(data => {
        setContent(data);
        setLoading(false);
      });
  }, [userId]);

  if (loading) return <div>Loading today's edge...</div>;

  return (
    <div className="daily-edge">
      <div className="header">
        <span className="streak">üî• {content.streak?.current_streak || 0} Day Streak</span>
        <h2>Today's Focus: {content.belief_display}</h2>
      </div>

      {content.content.insight && (
        <div className="card insight">
          <h3>üí° Key Insight</h3>
          <p>{content.content.insight.content}</p>
          <cite>‚Äî {content.content.insight.source}</cite>
        </div>
      )}

      {content.content.challenge && (
        <div className="card challenge">
          <h3>üéØ Today's Challenge</h3>
          <p>{content.content.challenge.content}</p>
          <button onClick={() => completeChallenge(content.content.challenge.id)}>
            Mark Complete
          </button>
        </div>
      )}

      {content.content.iconic_story && (
        <div className="card story">
          <h3>‚≠ê Iconic Insight</h3>
          <p>{content.content.iconic_story.content}</p>
          <cite>‚Äî {content.content.iconic_story.source}</cite>
        </div>
      )}

      {content.suggested_scenario && (
        <div className="cta">
          <p>Practice today's belief with role-play:</p>
          <button onClick={() => startScenario(content.suggested_scenario.id)}>
            Start: {content.suggested_scenario.name}
          </button>
        </div>
      )}
    </div>
  );
}
```

## Adding Your Own Content

### From Google Drive

Create a folder called `Daily Edge/` in your training content folder with subfolders:
- `Fulfilment/`
- `Control/`
- `Resilience/`
- `Influence/`
- `Communication/`

The Google Drive sync will automatically import these as Daily Edge content.

### Manual Import

```bash
curl -X POST http://localhost:3001/api/daily-edge/content \
  -H "Content-Type: application/json" \
  -d '{
    "belief": "resilience",
    "content_type": "quote",
    "title": "Custom Quote",
    "content": "Your quote text here...",
    "source": "Speaker Name",
    "tags": ["custom", "motivation"]
  }'
```

## Gamification Features

### Streaks
- Users build streaks by engaging daily
- Current streak resets if they miss a day
- Longest streak is preserved
- Displayed on leaderboard

### Challenge Completion
- Challenges can be marked complete
- Users can add reflections
- Progress tracked per belief system

### Leaderboards
- By streak length
- By challenges completed
- By total engagement

## Best Practices

1. **Consistency**: Encourage agents to check Daily Edge at the same time each day (e.g., morning routine)

2. **Reflection**: Have agents write reflections on challenges - this deepens learning

3. **Integration**: After Daily Edge, suggest a related role-play scenario

4. **Team Discussion**: Use weekly meetings to discuss the belief of the week

5. **Recognition**: Celebrate streak milestones (7 days, 30 days, etc.)
