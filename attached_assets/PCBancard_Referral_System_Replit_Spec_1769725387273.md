# Referral Management System â€” Replit Build Specification

## Overview

Build a role-based referral management system where sales agents can track and communicate with their referrals, managers can oversee their team's referrals, and admins have full visibility across the organization.

---

## User Roles & Permissions

| Role | Can See | Can Add | Can Edit | Can Delete |
|------|---------|---------|----------|------------|
| **Agent** | Own referrals only | Yes (to self) | Own referrals | Own referrals |
| **Manager** | Team referrals (all agents they manage) | Yes (assign to self or team) | Team referrals | Team referrals |
| **Admin** | All referrals system-wide | Yes (assign to any agent) | All referrals | All referrals |

---

## Data Model

### Referral Schema

```javascript
const ReferralSchema = {
  id: string,                    // Unique identifier
  
  // Referral Source (WHO referred them)
  referredBy: {
    type: 'agent' | 'merchant' | 'external',
    agentId: string | null,      // If referred by an agent
    merchantId: string | null,   // If referred by existing merchant
    externalName: string | null, // If external (friend, family, etc.)
    externalRelation: string,    // "Friend", "Family", "Business Partner", etc.
  },
  
  // Assigned Agent (WHO owns this referral)
  assignedAgentId: string,       // The agent working this referral
  assignedAgentName: string,     // Denormalized for display
  
  // Referral Contact Info
  businessName: string,
  contactName: string,
  email: string,
  phone: string,
  address: {
    street: string,
    city: string,
    state: string,
    zip: string,
  },
  
  // Business Details
  businessType: string,          // "Restaurant", "Retail", "Service", etc.
  estimatedMonthlyVolume: number,
  currentProcessor: string,
  
  // Status Tracking
  status: 'NEW' | 'CONTACTED' | 'APPOINTMENT_SET' | 'PROPOSAL_SENT' | 'NEGOTIATING' | 'WON' | 'LOST',
  temperature: 'HOT' | 'WARM' | 'COLD',
  
  // Communication
  notes: [{
    id: string,
    text: string,
    createdBy: string,
    createdAt: timestamp,
  }],
  
  lastContactedAt: timestamp,
  nextFollowUpDate: date,
  
  // Metadata
  createdAt: timestamp,
  updatedAt: timestamp,
  createdBy: string,             // User who created the record
}
```

## Feature 4: Search Implementation

### Searchable Merchant Selector Component

```jsx
// Component: MerchantSearch
// Used when selecting "referred by merchant"

const MerchantSearch = ({ userRole, userId, managerId, onSelect }) => {
  const [query, setQuery] = useState('');
  const [results, setResults] = useState([]);
  
  // API call varies by role
  const searchMerchants = async (searchTerm) => {
    let endpoint;
    
    switch (userRole) {
      case 'agent':
        // Only search this agent's merchants
        endpoint = `/api/merchants?agentId=${userId}&search=${searchTerm}`;
        break;
      case 'manager':
        // Search all merchants for agents under this manager
        endpoint = `/api/merchants?managerId=${userId}&search=${searchTerm}`;
        break;
      case 'admin':
        // Search all merchants system-wide
        endpoint = `/api/merchants?search=${searchTerm}`;
        break;
    }
    
    const response = await fetch(endpoint);
    return response.json();
  };

  return (
    <div className="merchant-search">
      <input
        type="text"
        placeholder={
          userRole === 'agent' 
            ? "Search your merchants..." 
            : "Search merchants..."
        }
        value={query}
        onChange={(e) => {
          setQuery(e.target.value);
          if (e.target.value.length >= 2) {
            searchMerchants(e.target.value).then(setResults);
          }
        }}
      />
      
      {results.length > 0 && (
        <ul className="search-results">
          {results.map(merchant => (
            <li key={merchant.id} onClick={() => onSelect(merchant)}>
              <div className="merchant-name">{merchant.businessName}</div>
              <div className="merchant-contact">{merchant.contactName}</div>
              {/* Show agent name for managers/admins */}
              {userRole !== 'agent' && (
                <div className="merchant-agent">
                  Agent: {merchant.agentName}
                </div>
              )}
              <div className="merchant-volume">
                ${merchant.monthlyVolume.toLocaleString()}/mo
              </div>
            </li>
          ))}
        </ul>
      )}
    </div>
  );
};
```

### Search Fields

The merchant search should query across:
- `businessName` (primary)
- `contactName` 
- `merchantId` / `MID`
- `phone`
- `email`

Results should be ranked by:
1. Exact match on business name
2. Starts-with match on business name
3. Contains match on business name
4. Matches on other fields
